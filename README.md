### 一. 注意
1、关于音源

音质最高支持320k。

仅供在线试听，禁止批量下载！

批量下载会被封禁IP，请规范使用！

2、关于使用

musicfree请在设置里选择：播放失败时暂停！

否则在某些情况下会被封禁IP
