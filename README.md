# <p align="center">使用教程</p>


### 一. 注意
#### 1.1 关于音源

音质最高支持320k。

不支持数字专辑，反复请求可能会导致IP被封！

仅供在线试听，禁止批量下载！

批量下载会被封禁IP，请规范使用！

#### 1.2 关于使用

MusicFree请在设置中选择：`播放失败时暂停`

否则在某些情况下可能会导致被临时封禁IP
